Bootstrap for Beginners

What is Twitter Bootstrap

  • Front-End framework for Ui's and themes
  • HTML/CSS/Javascript
  • Used to building responsive, mobile first web applications and websites.
  • Most popular framework thus far.

Why Use Bootstrap

  • Increases development speed
  • Responsiveness
  • Consistency
  • Customizable, can add own classes as well.
  • Support, huge community and best documentation.
